# Legacy Artifacts

Esta carpeta contiene scripts históricos que no forman parte del flujo productivo de la API FastAPI.

- `cmcc_model.py`: versión antigua standalone de CMCC.
- `mcc_model.py`: versión antigua standalone de MCC.
- `function_calls_example.txt`: ejemplos manuales de ejecución para pruebas locales.

El código productivo actual está en:

- `app.py`
- `core/`
- `api/`
- `schemas/`
- `services/`
- `registry/`
- `models/`
- `utils/`
